# 💬 Real-Time Chat Application

Aplicação de chat em tempo real desenvolvida com foco em comunicação instantânea com websockets.

## 🚀 Visão Geral

Este projeto consiste em um sistema de chat em tempo real que permite a comunicação entre múltiplos usuários através de conexões persistentes.

A aplicação foi construída com o objetivo de explorar conceitos como:

- Comunicação em tempo real (WebSocket)
- Gerenciamento de conexões simultâneas
- Troca de mensagens instantânea
- Spring WebSocket (STOMP)

---

## ⚙️ Funcionalidades

- ✅ Conexão em tempo real entre usuários
- ✅ Envio e recebimento instantâneo de mensagens
- ✅ Múltiplos usuários conectados simultaneamente
- ✅ Atualização dinâmica sem reload da página
- ✅ Estrutura preparada para expansão (salas, autenticação, etc.)
